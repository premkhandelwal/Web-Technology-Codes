Spyware

1. Introduction

Any operation that leads to the misuse of system resources or system data is termed as cyber attack. Any software that performs a cyber attack by a self extracting script is termed as malicious application. These applications record the personal details without the knowledge of the user. This data could be manipulated by the attacker for attack execution. The attacker creates malware to distort the system operations. It acquires information such as passwords, individual personal details, bank account number, credit or debit pin numbers, by passive recording. The malware causes damage after entering into the target system by performing operations such as zipping the user details, logging the keys pressed etc.
